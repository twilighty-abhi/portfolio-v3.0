# MDX Features Demo

With MDX, you can mix Markdown with HTML and even React components!

## Line Breaks and Spacing

You can use HTML tags for precise control over spacing:

This is on a new line after a `<br />` tag.

## Custom HTML Elements

You can use any HTML elements:

<div style={{ padding: '20px', backgroundColor: '#f3f4f6', borderRadius: '8px', margin: '20px 0' }}>
  This is a custom styled div with padding and background color.
</div>

<blockquote style={{ borderLeft: '4px solid #3b82f6', paddingLeft: '16px', fontStyle: 'italic' }}>
  This is a custom styled blockquote.
</blockquote>
