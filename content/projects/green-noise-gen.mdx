# Green Noise Generator

A simple but effective tool that generates green noise to help with focus, sleep, and relaxation.

## Features

- Green noise generation with adjustable parameters
- Simple, intuitive user interface
- Customizable volume and tone
- Timer functionality for sleep sessions

## What is Green Noise?

Green noise is a variant of white noise with a power spectral density that is proportional to the frequency. It contains more energy in the middle frequency range, making it particularly soothing for human ears.

## Technologies

This project leverages the Web Audio API to generate high-quality green noise in real-time within the browser.
